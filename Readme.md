# PlayTimer
Author: amiq11 <smk.hhs[at]gmail.com>

## 概要
プロセスを起動している時間を測定します。
ゲームをしすぎる人や、ブラウザを立ち上げている時間を測るといったことにお使いください。

## インストール方法
解凍して適当な場所に置くだけです。

MSVCR110.dllが見つかりません、といったエラーが出る場合は、以下のURLからMicrosoft Visual C++ 2012再配布パック(x86)をインストールしてください。
http://www.microsoft.com/ja-jp/download/details.aspx?id=30679

## 使い方
上のテキストボックスに入力されているプロセスの起動している時間を測定します。（デフォルトではFF14(ffxiv.exe)になっています）
測定方法としては、1秒に1回プロセス一覧を取得し、指定されたプロセス名が存在するか確認する、という方式をとっています。
また、時間は名前ごとに分けて保存しているので、複数のプログラムを使い分けることができます。
終了時に指定していたプロセスも保持しており、再度起動した際にはすぐに測定が開始できます。
